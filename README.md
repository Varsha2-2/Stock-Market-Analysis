# Stock-Market-Analysis
Objective

Structure and analyze stock market data

Track key financial and trading KPIs

Deliver interactive dashboards for decision-making

Tools Used

Excel | SQL (MySQL) | Tableau | Power BI

Workflow

Data cleaning and KPI identification in Excel

Normalized database design and analysis using SQL

Interactive dashboards built in Tableau and Power BI

KPI validation through SQL query cross-checks

Key KPIs

Market Capitalization, Trading Volume, Portfolio Value & Return, Volatility, Sharpe Ratio, Dividend Yield, Trade Win Rate, Order Execution Rate, Trader P&L

Deliverables

Excel, Tableau, and Power BI dashboards with validated KPIs and insights.
